Exercise: Introduction to GitHub

## My First Application

This repository contains a simple web application that demonstrates building applications using HTML, CSS, and JavaScript.

### Features

- **Interactive UI**: Click counter functionality
- **Responsive Design**: Works on desktop and mobile devices
- **Modern Styling**: Clean, gradient-based design
- **JavaScript Interactivity**: Dynamic content updates

### Getting Started

To run this application:

1. Clone this repository
2. Open `index.html` in your web browser
3. Or use a local server:
   ```bash
   # Using Python 3
   python -m http.server 8000
   
   # Using Node.js
   npx http-server
   ```
4. Navigate to `http://localhost:8000` in your browser

### Application Structure

- `index.html` - Main HTML structure
- `styles.css` - Styling and layout
- `script.js` - JavaScript functionality

### What You Can Do

- Click the "Click Me!" button to increment the counter
- Click "Reset" to reset the counter to zero
- Resize your browser to see the responsive design in action

### Learning Outcomes

This project demonstrates:
- Basic HTML5 structure
- CSS styling and animations
- JavaScript DOM manipulation
- Event handling
- Responsive web design principles
